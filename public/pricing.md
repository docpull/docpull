# docpull Pricing

## Pay-per-page

| Pages | Cost (USDC) |
|-------|-------------|
| 1 | $0.001 |
| 10 | $0.010 |
| 50 | $0.050 |
| 100 | $0.100 |
| 1,000 | $1.000 |

Price: $0.001 USDC per page.
No subscriptions. No minimums. No API keys.

## Payment method

Payments are made in USDC on Base mainnet (eip155:8453) via the x402 v2 protocol.
Use the free /probe endpoint to check page count and exact cost before paying.

## Free endpoints

- GET /health — always free
- GET /probe — always free
